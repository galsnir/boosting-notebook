# ML Boosting Notebook

Homework 2 (Boosting) — Course 3945, 2025A.

This repository contains a Jupyter notebook implementing boosting algorithms (AdaBoost, Gradient Boosting, Random Forest comparisons) on synthetic datasets.

## Contents

- `Boosting.ipynb` — main notebook with all exercises and results.

## Requirements

- Python 3.9+
- `numpy` >= 1.15.4
- `scikit-learn`
- `matplotlib`
- `jupyter`

Install with:

```bash
pip install -r requirements.txt
```

## Usage

```bash
jupyter notebook Boosting.ipynb
```
